CORS service

Webscraping service to convert target HTML elements to JSON

Usage:
  1. http:///[path]?url=[url]
  2. path: ‘/’ seperated list of tags and modifiers
  3. url: urlencoded source url
Tag Path Modifiers
  • [x[,y]]: Processes tags in range of x[ ->y]
  • {tag, tag, tag’…}’: processes tags with initial result set
  • ‘.,#,[]’: ‘CSS selectors’
keywords
  • sort: ‘Sorts Results by content in ascending order’,
  • clean: ‘Strips all HTML from result items contents’,
  • .json: ‘Return results as json’

Notes:

All hrefs and img srcs are converted to absolute urls

Examples:

  1. Headlines From NYT homepage:
    http:///h2.story-heading/a.json?url=https://www.nytimes.com/

  2. Headlines From NYT homepage column 1:
    http:///div.a-column/h2.story-heading/a.json?url=https://www.nytimes.com/

  3. 3rd Headline From NYT homepage OP-ED Contributor list:
    http:///div.opinion-c-col-right-region/div/ul/li/h2.story-heading/a.json/[1,2]?url=https://www.nytimes.com/
